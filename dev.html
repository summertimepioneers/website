Метки и переходы

Переход к метке Ren'Py
Команда jump немедленно переходит к указанной метке. Команда call переходит и возвращается обратно после окончания блока. 
label start:
    "Добро пожаловать в Совёнок."
    jump intro_scene          # безвозвратный переход

label intro_scene:
    scene bg summer_camp
    "Лето. Всё только начинается."
    call show_characters      # переход с возвратом

label show_characters:
    show eileen happy
    return                    # возврат к месту вызова

Переменные

Объявление и использование переменных Ren'Py
Переменные объявляются в блоке init python или через default. Используйте их для хранения состояния (флаги, счётчики, имена персонажей). 
default player_name = "Семён"   # имя игрока
default affection = 0           # счётчик симпатии

label day_one:
    $ affection += 1            # изменяем переменную
    "[player_name], доброе утро!"
    if affection >= 5:
        "Кажется, кто-то тебе доверяет."

Переходы

Встроенные и кастомные переходы между сценами и экранами.

Dissolve

Плавное растворение сцены Ren'Py
dissolve — базовый переход, встроен в Ren'Py. Второй параметр — длительность в секундах. 
scene bg forest with dissolve         # стандартный dissolve

# Кастомная скорость:
define slow_dissolve = Dissolve(2.0)  # 2 секунды
scene bg night with slow_dissolve

Кастомные переходы

ImageDissolve — маскированный переход Ren'Py
ImageDissolve использует чёрно-белую маску: белые пиксели появляются первыми, чёрные — последними. Отлично подходит для эффектов «разлита чернила», «снег», «вспышка». 
# Маска — изображение masks/wipe_left.png
define wipe = ImageDissolve("masks/wipe_left.png", 1.0)

scene bg classroom with wipe

Трансформы

Позиционирование, движение и масштабирование объектов на экране.

Движение

ATL-анимация — движение спрайта Ren'Py
ATL (Animation and Transformation Language) позволяет описывать анимации прямо в коде. Блок transform описывает как объект движется, появляется, исчезает. 
transform slide_in_left:
    xpos -300 ypos 0.8        # начальная позиция за экраном
    linear 0.6 xpos 0.3       # плавный сдвиг вправо за 0.6с

transform slide_in_right:
    xpos 1300 ypos 0.8
    linear 0.6 xpos 0.7

# Применение:
show eileen happy at slide_in_left

Масштаб

Zoom — приближение / отдаление Ren'Py
zoom масштабирует объект от центра. 1.0 — нормальный размер, 2.0 — вдвое больше. Комбинируйте с linear для анимации. 
transform zoom_in:
    zoom 0.8 alpha 0
    linear 0.5 zoom 1.0 alpha 1.0   # появление с увеличением

show eileen happy at zoom_in

Работа со спрайтами, диалоговым окном, фонами и CG-иллюстрациями.

Спрайты

Объявление и вывод спрайта Ren'Py
Спрайты объявляются через image или папочную систему Ren'Py. Показываются командой show, убираются hide. 
# Объявление спрайта (обычно в отдельном файле characters.rpy)
image masha happy = "sprites/masha/happy.png"
image masha sad   = "sprites/masha/sad.png"

# Показ и смена состояния
label scene_example:
    show masha happy at center
    "Маша улыбается."
    show masha sad
    "Но вдруг погрустнела."
    hide masha with dissolve

Диалоговое окно

Кастомизация textbox через screens.rpy Ren'Py
Диалоговое окно определяется экраном say в файле screens.rpy. Меняйте background, xsize, ypos для кастомного вида. 
screen say(who, what):
    window:
        id "window"
        background "gui/textbox.png"   # ваш фон диалога
        xsize 1280
        ypos  0.75                     # позиция по вертикали (0-1)
        padding (30, 20)

        vbox:
            if who:
                text who id "who" style "say_label"
            text what id "what" style "say_dialogue"

Фоны (BG)

Показ фона и смена времени суток Ren'Py
Фоны показываются командой scene. После scene можно сразу добавить переход. Для смены времени суток удобно использовать отдельные файлы или LayeredImage. 
image bg camp_day   = "backgrounds/camp_day.png"
image bg camp_night = "backgrounds/camp_night.png"

label evening:
    scene bg camp_day with dissolve
    "Тёплый летний день..."
    scene bg camp_night with Dissolve(1.5)
    "Наступила ночь."

CG-иллюстрации

Показ CG и добавление в галерею Ren'Py
CG — полноэкранные иллюстрации. Показываются через scene без фоновых изображений. Для галереи используйте встроенный класс Gallery. 
# Показ CG
label cg_scene:
    scene black with fade
    scene cg/summer_end with dissolve
    "Последний вечер в лагере."
    scene black with fade

# Добавление в галерею (в init-блоке)
init python:
    g = Gallery()
    g.button("summer_end")
    g.image("cg/summer_end")
    g.locked_button("summer_end_lock")
