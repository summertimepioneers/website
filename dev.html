Привет, мододелы! 👋

Добро пожаловать в «Готовые пресеты кода для модов «Бесконечного лета»».

Это методичка специально для тех, кто создаёт моды. Здесь собраны проверенные, готовые блоки кода — всё, что чаще всего требуется в работе.

Вам больше не придётся каждый раз писать одно и то же с нуля. Просто копируете нужный пресет → вставляете → адаптируете под свой мод и продолжаете творить.

  • Для новичков — всё максимально понятно и с примерами.
  • Для опытных — быстрый доступ к уже отработанным решениям, чтобы не тратить время зря.

Методичка будет постоянно пополняться. Если найдёте полезный кусок кода, который стоит сюда добавить — пишите, обязательно включим.

Удачи в модостроении и пусть код всегда работает с первого раза! ✨

⚠️ Данные коды были написаны на Ren'Py версии 8.3.4. На некоторых версиях ниже код может не работать.

Начало пути

Самые первые шаги при создании мода — структура папок, инициализация и регистрация в меню игры.

1. Первый файл — init и регистрация мода в меню Ren'Py
Начало создания любого мода. Это самый первый код, который нужно написать. Он регистрирует ваш мод в списке модов «Бесконечного лета», подключает шрифты и включает режим разработчика.
📁 Где размещать этот код:
  1. Создайте папку мода: game/mods/(ваше_название)/
    Замените (ваше_название) на короткое имя БЕЗ ПРОБЕЛОВ, желательно латиницей — например: my_mod, summer_story
  2. Внутри этой папки создайте файл script.rpy
  3. Откройте файл и вставьте код в самом начале (до любых label, scene и т.д.)
⚠️ Важно: Везде в коде ниже замените (ваше_название) на одно и то же короткое имя вашего мода. Также замените (Название вашего мода) на то, как мод будет отображаться в меню (можно с пробелами и русскими буквами). 
# =============================================
#          ИНИЦИАЛИЗАЦИЯ МОДА
# =============================================
init:
# Пути к шрифтам (файлы шрифтов должны лежать по этим путям)
define (ваше_название)_font_path = "mods/(ваше_название)/(ваше_название)_fonts/(ваше_название)_font.ttf"

#   ⇩⇩⇩      ≍≍≍≍≍≍≍≍≍≍≍≍≍≍≍≍≍≍       ⇩⇩⇩
# УДАЛИТЬ ПЕРЕД ВЫПУСКОМ МОДА!
# Включает режим разработчика (F1 — консоль, быстрый перезапуск и т.д.)
$ config.developer = True

# Стиль названия мода в списке модов
style menu_(ваше_название) is default:
    font (ваше_название)_font_path      # какой шрифт использовать
    color "#b8860b"                     # цвет текста (золотой, можно поменять)
    size 46                             # размер текста

# Регистрация мода в главном меню игры
# Без этой строки мод не появится в списке!
$ mods["menu_(ваше_название)"] = "{=menu_(ваше_название)}(Название вашего мода){/=mod_label}"

Функции

Базовые конструкции Ren'Py: метки, условия, переменные и вызовы сцен.

Метки и переходы Ren'Py
jump — немедленный переход к метке без возврата.
call — переход с возвратом обратно после окончания блока. 
label start:
    "Добро пожаловать в Совёнок."
    jump intro_scene          # безвозвратный переход

label intro_scene:
    scene bg summer_camp
    "Лето. Всё только начинается."
    call show_characters      # переход с возвратом

label show_characters:
    show eileen happy
    return                    # возврат к месту вызова

Переменные Ren'Py
Переменные объявляются через default или в блоке init python. Используйте для флагов, счётчиков, имён персонажей. 
default player_name = "Семён"   # имя игрока
default affection = 0           # счётчик симпатии

label day_one:
    $ affection += 1            # изменяем переменную
    "[player_name], доброе утро!"
    if affection >= 5:
        "Кажется, кто-то тебе доверяет."

Переходы

Встроенные и кастомные переходы между сценами и экранами.

Dissolve — плавное растворение Ren'Py
dissolve — базовый переход, встроен в Ren'Py. Второй параметр — длительность в секундах. 
scene bg forest with dissolve         # стандартный dissolve

# Кастомная скорость:
define slow_dissolve = Dissolve(2.0)  # 2 секунды
scene bg night with slow_dissolve

ImageDissolve — маскированный переход Ren'Py
ImageDissolve использует чёрно-белую маску: белые пиксели появляются первыми. Подходит для эффектов «чернила», «снег», «вспышка». 
define wipe = ImageDissolve("masks/wipe_left.png", 1.0)
scene bg classroom with wipe

Трансформы

Позиционирование, движение и масштабирование объектов на экране.

ATL — движение спрайта Ren'Py
ATL (Animation and Transformation Language) позволяет описывать анимации прямо в коде. 
transform slide_in_left:
    xpos -300 ypos 0.8
    linear 0.6 xpos 0.3   # плавный сдвиг вправо за 0.6с

show eileen happy at slide_in_left

Zoom — приближение / отдаление Ren'Py
zoom масштабирует объект от центра. 1.0 — нормальный размер, 2.0 — вдвое больше. 
transform zoom_in:
    zoom 0.8 alpha 0
    linear 0.5 zoom 1.0 alpha 1.0   # появление с увеличением

show eileen happy at zoom_in

Визуал

Работа со спрайтами, диалоговым окном, фонами и CG-иллюстрациями.

Спрайты — объявление и вывод Ren'Py
Спрайты объявляются через image. Показываются командой show, убираются hide. 
image masha happy = "sprites/masha/happy.png"
image masha sad   = "sprites/masha/sad.png"

label scene_example:
    show masha happy at center
    "Маша улыбается."
    show masha sad
    "Но вдруг погрустнела."
    hide masha with dissolve

Диалоговое окно — кастомизация Ren'Py
Диалоговое окно определяется экраном say в файле screens.rpy. 
screen say(who, what):
    window:
        id "window"
        background "gui/textbox.png"
        xsize 1280
        ypos  0.75
        padding (30, 20)
        vbox:
            if who:
                text who id "who" style "say_label"
            text what id "what" style "say_dialogue"

Фоны (BG) — показ и смена Ren'Py
Фоны показываются командой scene. После scene можно сразу добавить переход. 
image bg camp_day   = "backgrounds/camp_day.png"
image bg camp_night = "backgrounds/camp_night.png"

label evening:
    scene bg camp_day with dissolve
    "Тёплый летний день..."
    scene bg camp_night with Dissolve(1.5)

CG-иллюстрации — показ и галерея Ren'Py
CG — полноэкранные иллюстрации. Для галереи используйте встроенный класс Gallery. 
label cg_scene:
    scene black with fade
    scene cg/summer_end with dissolve
    "Последний вечер в лагере."
    scene black with fade

init python:
    g = Gallery()
    g.button("summer_end")
    g.image("cg/summer_end")
